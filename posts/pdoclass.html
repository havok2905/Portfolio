What is Regulator?

Regulator: PHP PDO Class

Regulator is a PHP class wrapper for PDO, a prepared statement library for PHP. This class was built to make simple database queries to speed up development for more simple tasks.

What can Regulator Currently do?

As of this date, Regulator can perform the following functions:

  1. Make a database connection
  2. Perform a select query and return an array
  3. Perform an insert query
  4. Perform an update query

Why I chose to Start Regulator

I chose to start building Regulator in order to better understand prepared statements and to protect myself from SQL injections. Having to dissect the syntax bit by bit forced me to memorize the material.

Future Goals for Regulator

  1. Add a method for general queries by passing a string as an argument
  2. Expand on selectors and modifiers for existing query methods
