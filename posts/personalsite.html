What is my Personal Website?

Personal Website Github

My personal website is a collection of my work, a functioning blog system, an email client, as well as the accumulation of all of my personal branding efforts .

What can my Personal Site Currently do?

As of this date, my personal site can perform the following functions:

  1. Display current work
  2. Promote personal brand
  3. Allow for users to comment on blog entries
  4. Allow for users to send messages to me through a form

Why I chose to Start my Personal Site

I chose to start building my personal website in order to promote myself, my work, seek new clients and work opportunities, and to communicate with the rest of the web community through a blog. This site also allowed me to experiment with adaptive and responsive web design, with this site supporting three different views.

Future Goals for my Personal Site

  2. Classify Blog Functionality
  3. Classify Date Functionality
  4. Support Nested Comments in Blog
