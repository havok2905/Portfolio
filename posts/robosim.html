What is RoboSim?

RoboSim: Try JavaScript with Robots

Robosim is an HTML5 based learn JavaScript learning toll being developed within the Center for Distributed Learning at the University of Central Florida. I approached this project with the purpose of trying to figure out a way to teach programming to students who were more visually inclined. The application has a code editor, a stage for the robot to move, and a library of functions associated with the robot.

What can RoboSim Currently do?

As of this date, RoboSim can perform the following functions:

  1. Draw a moveable robot on a canvas stage
  2. Be controlled through custom JavaScript Functions
  3. Take in most JavaScript syntax
  4. Abstracts out HTML5 JavaScript syntax from controls to avoid confusing the user
  5. Has a fully functioning collapsible function library for reference

Why I chose to Start RoboSim

I chose to start RoboSim out of watching artistically inclined students struggle in math heavy computer science course. The issue was not that they could not think logically, but rather that abstract concepts in mathematics are far from what they are used to. This application hopes to aid in these students being able to visually see the results of their code as well as to possibly spark interests in game design, web design, and other artistically saturated technological careers.

Future Goals for RoboSim

  2. Support mechanical robot extensions and sensors for use in code challenges
