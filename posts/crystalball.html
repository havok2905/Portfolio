Crystal Ball App

What is the Crystal Ball App?

The Crystal Ball App is an application being developed while following Treehouse tutorials on learning Android development. It is an application which allows you to shake your phone for a random answer to a question that you ask it.

What can the Crystal Ball App Currently do?

As of this date, the Crystal Ball App can perform the following functions:

  1. Display size appropriate graphics
  2. Generate random answers
  3. Accelerometer controls

Why did I Start Crystal Ball App

After learning about responsive design and mobile application development through my Web Workshop course, I wanted to branch out to native app development with Android.
