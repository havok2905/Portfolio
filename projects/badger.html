Badger: Badge Distribution Program

What is Badger?

Badger has been in two different phases of development. The first phase was a rapid prototype badging portal for distributing badges through email for a small number of faculty taking part in an online course. The second phase of development for badger aims to make an all purpose set of badging tools for development. Badger is a project for the University of Central Florida and unfortunatley I can not display source code.

What can Badger currently do?

As of this date, Badger can perform the following functions:

  1. Rapid prototype can distribute badges to a user's Mozilla open badges backpack
  2. Create new Badges through a set of admin tools
  3. Pull badges from a database through a custom REST API
  4. Generate developer keys for developers using Badger API

What is the purpose of Badger

Badger began in development due to the rising use of badging systems in online learning. Gamification has played a role in enriching online educational environments and Badger was a small step in exploring this method of teaching.

Future Goals for Badger

  1. Support student and faculty login
  2. Log student progress through badging program
  3. Send email to student alerting badge award to both persona and misc backpacks
  4. Use of custom backpacking portal for badge viewing

Technology used in Badger

  • FuelPHP
  • Mozilla Open Badges
  • JavaScript with jQuery
