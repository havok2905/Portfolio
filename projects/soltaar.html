Soltaar

What is Soltaar?

https://github.com/havok2905/DIG4104C_P2

Soltaar is a visual impairment accessible memory matching game being developed as a final group project in accessibility for Web Workshop at UCF. The game will support users who are blind, color blind, and have blurred vision. This projects highlights developing around use of a screen reader without intruding on the average, non impaired, user's experience.

What can Soltaar do?

As of this date, Soltaar can perform the following functions:

  1. Play a matching game
  2. Flip cards using CSS3 3D transforms
  3. Navigatable with a screen reader
  4. Abstracts card order from users using screen reader

Why we chose to Start the Blog Reader

As of now there are not many blind accessible games available for play through browsers. We wanted to bring this experience to visually impaired users. Also, because this project uses secrecy of information as a primary game mechanic, this project allows us to explore many methods of information abstraction in the application's markup.

Future Goals for Soltaar
