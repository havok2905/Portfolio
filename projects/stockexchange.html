Stock Exchange

What is the Stock Exchange?

The Stock Exchange was a group project for Web Design Workshop at the University of Central Florida. Using a professor's api, we developed a responsive front-end mobile web application for students to use in a stock market game. Because of the redundant nature of the API that was provided to use, our group also used it as an exercise in object oriented development, resulting in us abstracting large chunks of code. This resulted in a huge relief to our JavaScript Developers.

What can the Stock Exchange Currently do?

As of this date, the Stock Exchange can perform the following functions:

  1. View Transaction Information
  2. Send money and stock to other users
  3. Update a personal profile picture
  4. View current funds

What Were my Roles in the Stock Exchange

I was the acting project manager throughout the duration of the project and PHP developer. I took part in abstracting the professor's API, building customizable profiles, building a database class which turned into Regulator, and managed other developers taking part on the team.
