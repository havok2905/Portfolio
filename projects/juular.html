Juular: jQuery Lightbox Plug-in

What is Juular?

Juular: jQuery Lightbox Plugin

Juular is a jQuery lightbox plug-in that I began to develop for use in future projects and to learn more about jQuery authoring. This project taught me about automatically executing functions, JavaScript prototypes, and jQuery as a framework.

What can Juular Currently do?

As of this date, Juular can perform the following functions:

  1. Construct a lightbox
  2. Take lightbox content from external HTML and PHP files
  3. Allow post data to be sent to PHP files
  4. Customizable in size and style

Why I chose to Start Juular

I chose to start building Juular out of a need to learn more about jQuery and how it works as a framework. Until this point I have been blindly using it with no real concept of how it functioned. Learning how to chain functions, use prototypes, and more will be helpful in future programming endeavors.

Future Goals for Juular

  1. Support full responsive behavior
  2. Allow for nested lightboxes
  3. Allow for moveable lightboxes
  4. Provide default themes for ease of use
