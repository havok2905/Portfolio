McLean Code Blog Reader

What is the McLean Code Blog Reader?

The McLean Code Blog Reader is an Android app I am developing to provide a mobile view of my website's blog. This will entail developing a blog API, as well as the Android application itself.

What can the Blog Reader do?

As of this date, the blog reader can perform the following functions:

  1. Retrieve a JSON response from an API call
  2. Display those items in a list

Why I chose to Start the Blog Reader

I chose to start the blog reader in an attempt to practive Java software development after taking a course in Object Oriented Programming at UCF. I wanted to bridge my understanding of web development with Java software development.

Future Goals for Blog Reader

  1. Support the use of a Web View
  2. Build Blog API
